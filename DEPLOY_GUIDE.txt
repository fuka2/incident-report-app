==================================================
  事故報告書作成支援アプリ　Railway デプロイ手順
==================================================

【ファイル構成】
  incident-report-app/
  ├── app.py              ← Flaskバックエンド
  ├── templates/
  │   └── index.html      ← フロントエンド
  ├── requirements.txt    ← Pythonライブラリ
  ├── Procfile            ← Railway起動設定
  └── .env.example        ← 環境変数テンプレート

==================================================
STEP 1: GitHubにアップロード
==================================================
1. GitHub (https://github.com) にログイン
2. 「New repository」をクリック
3. リポジトリ名：incident-report-app
4. 「Create repository」をクリック
5. このフォルダのファイルをすべてアップロード
   （「uploading an existing file」をクリック）

==================================================
STEP 2: Railway にデプロイ
==================================================
1. https://railway.app にアクセス
2. 「Login with GitHub」でログイン
3. 「New Project」→「Deploy from GitHub repo」
4. 「incident-report-app」を選択
5. 自動デプロイが始まります（2〜3分）

==================================================
STEP 3: 環境変数を設定
==================================================
Railway の「Variables」タブで以下を追加：

  変数名                  値
  ──────────────────────────────────────────────
  ANTHROPIC_API_KEY      sk-ant-（取得したAPIキー）
  KINTONE_SUBDOMAIN      67zry
  KINTONE_APP_ID         219
  KINTONE_API_TOKEN      D3ftZbFIkRBFCYpySa48YPPojHhEPyhg0NgJfGxa

設定後、自動で再デプロイされます。

==================================================
STEP 4: URLを確認してアクセス
==================================================
Railway の「Deployments」タブに
  https://xxxx.railway.app
のようなURLが表示されます。
このURLを社内で共有してください。

==================================================
使い方
==================================================
1. URLをブラウザで開く
2. AIが「どの報告区分か？」と聞いてくる
3. 「事故です」「①」などと回答
4. 何があったか自由に話す（音声入力も可）
5. AIが不足情報を1つずつ確認してくれる
6. 右側に報告書プレビューが表示される
7. 「kintoneに保存する」ボタンで保存完了

==================================================
注意事項
==================================================
・APIキーは外部に漏れないよう管理してください
・Railway無料プランは月500時間まで利用可能
・音声入力はChrome/Edgeブラウザで利用できます
==================================================
